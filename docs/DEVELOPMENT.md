# Development

## Pre-requisites
```
npm install -g nodemon
```

## Install

```
yarn
```

## Compile

```
tsc
```

## Run in development mode

```
yarn dev
```

## Run tests

```
yarn test
```

## Production Build and Start

Compiles, builds and starts the production server.

```
yarn prod
```

## Start 

Starts the currently compiled version.

```
yarn start
```
